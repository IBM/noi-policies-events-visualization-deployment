## Responsibilities

Maintainers are responsible for:

- Reviewing and merging pull requests
- Responding to issues and questions
- Maintaining project documentation
- Ensuring code quality and security
- Managing releases and versioning
- Enforcing the Code of Conduct

## Becoming a Maintainer

Contributors who have made significant contributions to the project may be invited to become maintainers. Criteria include:

- Consistent high-quality contributions
- Deep understanding of the codebase
- Active participation in code reviews
- Commitment to the project's goals and values
- Adherence to the Code of Conduct

If you're interested in becoming a maintainer, please reach out to the current maintainers.
